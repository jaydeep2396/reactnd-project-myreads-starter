# MyReads Project

This project was built for the Udacity React Nanodegree Program. The purpose of the project is to demonstrate understanding of the basic structure and operation of a React-based app.

### Installation

* Install all project dependencies with `npm install`
* Start the development server with `npm start`

### Project Description

#### Home Page

Books are stored in 3 categories.
1. Currently Reading 
2. Want to Read
3. Read

To change a book's category or remove a book from the list, click the appropriate options in selection menu.
For adding new books click on the + button at the bottom. It will navigate the app to search page.

#### Search Page

Books can be searched and added to the selves by choosing the option from selection menu.

Note: The backend API is limited to a fixed set of search terms. See below for valid search options.
